# 📅 Master 智能公文與訊息日曆轉換器

這是一套專為學校行政人員、教師、公務機關與教育工作者設計的「智慧行程解析與行事曆轉換工具」。

完全免安裝、免註冊、免架設伺服器，只要打開網頁，就能將 LINE 訊息、公文內容、活動通知或螢幕截圖，自動整理成可直接匯入 Google Calendar、Apple Calendar、Outlook 的標準行事曆事件。

無論您是資訊組長、教務主任、行政教師，或是不熟悉程式操作的一般使用者，都能快速上手，立即減少人工抄寫日期、時間、地點與活動名稱的工作。

## 【如何運作？只要 3 個步驟】

1. 將 LINE 訊息、公文內容貼入系統，或直接上傳活動截圖。
2. 選擇「本地解析」或「Gemini AI 智慧解析」。
3. 確認系統產生的行程內容後，一鍵加入 Google Calendar 或下載 ICS 行事曆檔案。

系統會自動協助辨識：

* 活動名稱
* 日期
* 開始時間
* 結束時間
* 地點
* 聯絡資訊
* 原始訊息內容
* 公文主旨
* 多筆行程資料

## 🔐 核心特色：資料留在使用者自己的瀏覽器

本系統採用純前端架構，所有操作都在使用者自己的瀏覽器中完成。

### 本地解析模式

使用本地 JavaScript 規則進行文字辨識，不需要 API Key，也不需要將資料傳送到其他伺服器。

適合處理：

* 一般 LINE 行程通知
* 研習通知
* 開會通知
* 公文時間與地點
* 民國日期
* 西元日期
* 多筆活動內容

### Gemini AI 解析模式

當文字格式較複雜，或需要分析圖片、截圖與非結構化內容時，可由使用者自行輸入 Gemini API Key 進行智慧辨識。

API Key 僅儲存在使用者自己的瀏覽器 LocalStorage，不會預先寫入原始碼，也不會由開發者保管。

> 請勿將自己的 Gemini API Key 寫入公開的 `index.html` 或上傳至 GitHub。

## ✨ 系統亮點功能

### 1. LINE 訊息快速轉換行程

只要將群組中的活動通知、研習訊息、開會提醒或行政公告貼入系統，即可自動擷取時間、日期、地點與活動名稱。

不需要再逐欄複製，也不必反覆切換 LINE 與行事曆。

---

### 2. 公文開會通知智慧解析

針對臺灣學校與公務機關常見的公文格式進行設計，可辨識：

* 主旨
* 開會事由
* 開會時間
* 開會地點
* 主持人
* 承辦人
* 聯絡電話

特別適合教務處、學務處、總務處、教育處與各類行政單位使用。

---

### 3. 截圖與活動海報 AI 辨識

支援直接上傳圖片、拖曳圖片，或使用 `Ctrl + V` 貼上螢幕截圖。

可用來辨識：

* LINE 對話截圖
* 公文照片
* 活動海報
* 研習通知
* 會議行程表
* 手機截圖
* 圖片型公告

圖片辨識功能需使用 Gemini API Key。

---

### 4. 多筆行程自動拆分

當一段文字中包含多個日期或多場活動時，系統會嘗試自動拆分為多筆行事曆事件。

適合處理：

* 一週行程
* 系列研習
* 多日會議
* 多場活動通知
* 月份工作事項
* 行政工作排程

---

### 5. 一鍵匯出 ICS 行事曆檔案

系統可將行程輸出為標準 `.ics` 格式。

ICS 可匯入：

* Google Calendar
* Apple Calendar
* Microsoft Outlook
* iPhone 行事曆
* Android 行事曆
* 其他支援 ICS 的日曆系統

---

### 6. 一鍵加入 Google Calendar

完成行程解析後，可直接跳轉到 Google Calendar 新增活動頁面。

使用者只需再次確認內容，即可儲存至自己的 Google 行事曆。

---

### 7. 手機與電腦皆可使用

介面採響應式設計，支援：

* Windows
* macOS
* Chromebook
* iPhone
* iPad
* Android 手機
* Android 平板

手機版另提供快速跳轉與貼上優化，方便行政人員隨時處理行程。

---

### 8. 行程資料自動暫存

解析後的行程會保存在使用者目前瀏覽器的 LocalStorage。

重新整理頁面後，仍可繼續查看先前處理的事件。

> 若清除瀏覽器資料、更換裝置或使用無痕模式，暫存內容可能會消失。

### GitHub Pages 部署步驟

1. 將本專案 Fork 到您的 GitHub 帳號，或建立新的公開 Repository。
2. 將 `index.html`、`README.md` 與 `LICENSE` 上傳至儲存庫根目錄。
3. 進入 Repository 的 `Settings`。
4. 點選左側的 `Pages`。
5. 在 `Build and deployment` 中選擇 `Deploy from a branch`。
6. Branch 選擇 `main`。
7. Folder 選擇 `/ (root)`。
8. 按下 `Save`。

部署完成後，網站網址通常為：

```text
https://你的GitHub帳號.github.io/master-calendar-converter/
```

## 🔑 Gemini API Key 使用方式

1. 前往 Google AI Studio 建立自己的 API Key。
2. 開啟本系統。
3. 點擊右上角「Gemini API 金鑰」。
4. 貼上 API Key。
5. 點擊「儲存金鑰」。
6. 切換至 AI 模式。
7. 開始進行文字或圖片辨識。

### API Key 安全提醒

* 請勿將 API Key 直接寫入公開原始碼。
* 請勿將自己的 API Key 上傳到 GitHub。
* 請勿在公共電腦儲存私人 API Key。
* 請勿將付費 API Key 提供給不特定使用者共用。
* 使用完畢後，可在系統中清除 API Key。
* 使用 Gemini API 可能受到配額與費用限制。
* Gemini 模型名稱與 API 規格可能隨 Google 更新而調整。

## ⚠️ 使用注意事項

本系統提供的是輔助辨識工具，解析結果仍可能因文字格式、圖片品質或 AI 判讀而出現誤差。

匯入行事曆前，請務必再次確認：

* 日期是否正確
* 民國年與西元年是否正確
* 上午與下午是否正確
* 開始與結束時間是否正確
* 活動地點是否完整
* 是否有多筆活動遺漏
* 是否有文字辨識錯誤
